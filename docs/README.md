# Fragify - CS2 Stats Platform

Plataforma web de estadísticas para Counter-Strike 2.

## Características

- 📊 Estadísticas detalladas de partidas
- 👤 Perfiles de jugador con K/D, HS% y win rate
- 🏆 Leaderboards des Premier y Competitive
- 📱 Bot Telegram integrado
- 🔐 Autenticación Steam OpenID + email/password
- 🎛️ Panel de administración

## Stack Técnico

- **Frontend**: Next.js 15.5, TypeScript, TailwindCSS
- **Backend**: Next.js API Routes
- **Database**: TiDB Cloud
- **Hosting**: Vercel, Proxmox VE
